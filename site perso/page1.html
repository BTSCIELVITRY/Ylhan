L'aléatoire dans les jeux





Comment obtenir un nombre aléaoire ?

Créer une fonction pour avoir un nombre aléatoire est très simple. Pour cela, il suffit de faire un petit programme, dans le cas si-dessous en python
import random
# Générer un entier aléatoire entre 1 et 10
nombre_aleatoire = random.randint(1, 10)
print(f"Nombre aléatoire: {nombre_aleatoire}")

# Générer un nombre flottant aléatoire entre 0 et 1
nombre_flottant = random.random()
print(f"Nombre flottant: {nombre_flottant}")

On peut créer le meme algorithme avec beaucoup d'autre langage, comme le C, le Javascript, ou encore le C++ et le java.

l'aléatoire avec des dés

Coder un dé est très similaire avec le code au-dessus, il faut simplement générer un nombre qui représente une face du dé (cela peut etre un dé à 6 faces, à 20 faces etc...) import random

de_6_faces = random.randint(1, 6)
print(f"Résultat du lancer de dé à 6 faces : {de_6_faces}")

de_20_faces = random.randint(1, 20)
print(f"Résultat du lancer de dé à 20 faces : {de_20_faces}")

De plus, on peut lancer plusieurs dés à la fois
nombre_de_des = 3
lancers = [random.randint(1, 6) for _ in range(nombre_de_des)]
print(f"Résultats des {nombre_de_des} lancers : {lancers}")

Une forme d'aléatoire plus complexe : avec des cartes

Pour créer un jeu de carte aléatoire, il faut prendre plus de point en compte. En effet, il faut d'abord donner une valeur à chaque carte, cela est compris entre 1 et 10 dans les jeux de carte classique, avec d'autre carte spécial comme le valet ou la dame.
Ensuite, il faut donner une couleur aux cartes pour les différencier. Il faut aussi être en capacité de mélanger le jeu pour que le résultat change à chaque simulation du code.
import random
# Étape 1 : Créer un jeu de cartes
valeurs = ['2', '3', '4', '5', '6', '7', '8', '9', '10', 'Valet', 'Dame', 'Roi', 'As']
couleurs = ['Cœur', 'Carreau', 'Trèfle', 'Pique']
jeu_de_cartes = [f"{valeur} de {couleur}" for valeur in valeurs for couleur in couleurs]
# Étape 2 : Mélanger le jeu
random.shuffle(jeu_de_cartes)
# Étape 3 : Tirer une carte
carte_tiree = jeu_de_cartes.pop() # Retire une carte du dessus du paquet
print(f"Carte tirée : {carte_tiree}")
print(f"Cartes restantes : {len(jeu_de_cartes)}")
